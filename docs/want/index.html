Q.アイマストドンに技術的に貢献したいんだけど？

MastodonはAGPLライセンスのオープンソースのプロダクトです。アイマストドンのソースコードは https://github.com/imas/mastodon にあります。forkしてPull Requestをどうぞ。

また、iM@Study（アイマスハッカソン）のSlackのimastodonチャンネルでも技術的な話をしています。ご理解とご協力をよろしくお願いいたします。

Q.アイマストドンの機能に意見があるんだけど？技術的なことはわからないけど。

大歓迎です！@imastodon宛にリプライ（DMでも可）で教えてください。管理人（フサギコ）に伝わります。

もしくは@deflisか@YaQまで「あいましんきんぐ」に入りたい旨をお伝えいただくと、Discordで技術チーム含むたくさんのアイマストドンユーザーと機能について議論ができます。ご活用ください。

タイムラインで意見、不満などを発言するのは非推奨です。@imastodonへのリプライやあいましんきんぐをご利用ください。
